# 🚘 Driver Drowsiness Detection System

>  A real-time embedded safety system designed to detect driver fatigue and prevent accidents.

---

## 📌 Project Overview

The Driver Drowsiness Detection System is an embedded solution that monitors the driver's eye activity using an eye blink sensor. If the system detects that the driver's eyes are closed for a prolonged period, it identifies drowsiness and immediately triggers safety actions.

This system is designed to reduce road accidents caused by driver fatigue by providing real-time alerts and automatic vehicle control.

---

## 🎯 Objectives

- Detect driver drowsiness in real time  
- Alert the driver using a buzzer  
- Automatically stop the vehicle to prevent accidents  
- Build a low-cost and efficient safety system  

---

## 🔥 Features

- 👁️ Real-time eye blink monitoring  
- 🔊 Instant buzzer alert system  
- ⚙️ Automatic motor shutdown using relay  
- ⚡ Fast response time  
- 💰 Low-cost implementation  
- 🛠️ Easy to build and deploy  

---

## 🛠️ Components Used

- Arduino Uno

- Buzzer

## 💻 Technologies Used

- Embedded C (Arduino Programming)

## ⚙️ Working Principle
<table>
  <tr><td>
1. The eye blink sensor continuously monitors the driver's eye movement  <br>
2. If the eyes remain closed beyond a threshold time, drowsiness is detected   <br>
3. The Arduino processes the signal from the sensor   <br>
4. A buzzer is activated to alert the driver   <br>
5. The relay module cuts off the motor (simulating vehicle stop)  </td>
